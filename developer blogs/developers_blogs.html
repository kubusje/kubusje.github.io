Dear reader,

After some time off(working on school stuff) I am back.
And i finally worked on a feature that has been highly requested: skins.
In the main menu you can buy a skin with the previously added coins and then see the skin in game.
I am also going to change the version name system,
so instead of 0.1.09 it would be 0.1.90 so that is why this is version 0.2.00 and not 0.1.10.
The first number is the major release, the seconde the semi-major updates,
the third the "normal" update and the fourth cijfer is the minor update, for things like bug fixes.

  • Added skin selector in the main screen

  • Added WIP kubusje logo to startup
